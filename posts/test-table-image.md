## 3. 表格中含绝对值（管道符冲突测试）

以下测试 `$|x|$` 在表格中是否正常渲染。Markdown 表格的列分隔符也是 `|`，可能会与 LaTeX 绝对值符号产生冲突。

### 3a. 直接使用 `|` 表示绝对值

| 表达式         | 值               |
|---------------|-----------------|
| $\vert -3 \vert$ | 3            |
| $\vert 5 \vert$  | 5            |
| $\vert 0 \vert$  | 0            |

> 注意：这里使用了 `\vert` 而非裸 `|`，因为裸 `|` 在 Markdown 表格中会被误判为列分隔符。

### 3b. 直接使用裸 `|` 的绝对值（预期会出错）

| 表达式         | 值               |
|---------------|-----------------|
| $|-3|$        | 3               |
| $|5|$         | 5               |
| $|0|$         | 0               |

> ⚠️ 如果上表渲染异常（列错位、公式断裂），说明 Markdown 的 `|` 与 LaTeX 的 `|` 存在冲突。

---

## 4. 正文中的绝对值公式（不在表格中）

以下公式不在表格中，应该正常渲染：

- $|x|$ 表示 $x$ 的绝对值
- $|-3| = 3$
- $\vert x \vert$ 也是绝对值的写法

## 结论

- 表格需要 CSS 添加边框和背景才能正常显示（HTML 默认表格无样式）
- 图片需要 CSS `max-width: 100%` 防止大图撑出页面
- 在表格中使用 LaTeX 绝对值 `$|x|$` 时，应使用 `\vert` 代替裸 `|`，避免与 Markdown 表格列分隔符冲突
- 正文中的 `$|x|$` 不受影响，可正常使用
